Cara Kerja

  1. 10 pulls akan langsung di save oleh server ketika menekan tombol gacha
  2. Server pertama akan melakukan rolling rarity tier (1⭐, 2⭐, 3⭐)
  3. Setelah itu server baru akan melakukan rolling pada poll rarity tersebut
  4. Semua unit pada rarity tier yang sama akan memiliki probabilitas yang sama
  5. Apabila user tidak sengaja disconnect dari server, pull tidak akan hilang!
